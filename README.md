# NodeGroupingTestApplication
''分散システムにおけるノードのグルーピングとグループ内マスターノードによる分断耐性の強化''の実装

現状,グルーピングまでは正しく動作するがノードリストの配布が上手く行かないノードがある. 

## Usage
1. ```$ docker-compose build```  
1. ```$ docker-compose up -d```
